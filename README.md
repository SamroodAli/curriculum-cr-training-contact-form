## Built With

### Setup
1. Clone repository: 
```sh
  $ git clone https://github.com/SamroodAli/curriculum-cr-training-contact-form.git

```
2. Change directory inside project
```sh
cd curriculum-cr-training-contact-form
```
3. In your terminal, run 
```sh
npm i
```
